svelte-box playground

Five live demos. Each page focuses on one container so the behavior is easy to read.

/box

Proxy-backed. Transparent property forwarding (box.foo reads the inner object), callable when wrapping a function, instanceof Box through subclasses. Bottom of the page shows clone, box.toConst(), and isBox.

/fastbox

Plain class, no proxy. Same .value reactivity, same helpers and type guards. Faster to construct, no transparent forwarding. Reach inner properties through .value.

/const

Read-only views. ConstBox and ConstFastBox in both capture mode (independent snapshot) and borrow mode (shared live state with the source). Side-by-side mutation makes the difference visible.

/lazy

Deferred-loader cell. The loader runs on the first prefetch(); later calls return the cached promise until reset() clears it. The demo counts loader invocations so the cache behavior is verifiable.
